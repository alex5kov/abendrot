1. Setup SAP NetWeaver Cloud

Go to SAP NetWeaver Cloud Developer Center, get your free developer license, and perform all the steps covered under the "Step by step" section.

2. Install Maven and Git in Eclipse

SAP NetWeaver Cloud is build technology and source code management system agnostic, hence these two topics are not covered in the SAP step by step guide from section 1. Abenrot uses maven as build technology and git as source code management system; Update your eclipse environment to support maven and git:

Maven Integration for WTP (m2e-wtp) : go to https://www.eclipse.org/m2e-wtp/. Follow the installation procedure at https://www.eclipse.org/m2e-wtp/download/. Install the latest release.

Command line maven (optional, but strongly encouraged) : go to http://maven.apache.org/. Follow the installation procedure at http://maven.apache.org/download.html. Install the latest release.

Eclipse Git (egit): go to http://www.eclipse.org/egit/. Follow the installation procedure at http://www.eclipse.org/egit/download/. Install the latest release.

3. Create a Github Repository

Abendrot is a tutorial on how to write a SAP NetWeaver Cloud application. You will use a github repository to host and monitor your work. You will also clone abendrot itself to another directory. And you will use a good compare tool to diff the two directories at various stages of the tutorial.

Setup a github account, if you have not already one.

Log to your account at https://github.com/<your account>

Create an empty github repository. The place to do so is not obvious for a github beginner, it is this icon on the top right toolbar.

Check 'Initialize this repository with a README' for the very reason that github gives you : 'This will allow you to git clone the repository immediately' and choose the maven .gitignore.

Create and clone immediately your repository. You'll get something like this on your box:

  <repository name>/
  ├── .gitignore
  └── README.md

Clone abendrot from https://github.com/cthiebaud/abendrot.git, and checkout the initial state of abenrot (i.e. the first commit). Using the git command line :

$ git clone https://github.com/cthiebaud/abendrot.git  
$ cd abendrot
$ git checkout 4d427b99f3b60947e3535e045cdf4b35f979ed7f
